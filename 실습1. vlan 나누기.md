문제 
1. 현재 네트워크는 192.168.10.0/24 야. 이 네트워크를 7개 vlan으로 나누어라.
2. 각 네트워크 이름은 vlan1~7까지 주고
3. 각 vlan의 vlan명,CIDR, 네트워크마스크, 네트워크아이디, 호스트ip 범위, 호스트 개수를 표로 만들어라.

### 기술적 유의사항

* **네트워크 ID**: 해당 서브넷의 첫 번째 주소로, 네트워크 자체를 식별하는 데 사용되며 호스트에 할당할 수 없습니다.
* **브로드캐스트 IP**: 해당 서브넷의 마지막 주소로, 네트워크 내의 모든 호스트에게 데이터를 전송할 때 사용됩니다.
* **Default Gateway**: 일반적으로 각 VLAN의 가용 IP 범위 중 첫 번째 주소(예: VLAN 1의 경우 192.168.10.1)를 게이트웨이 주소로 설정하여 라우팅을 수행합니다.
* **서브넷 마스크 연산**: 256(전체) / 8(서브넷 개수) = 32(증분값)입니다. 각 네트워크 ID는 32씩 증가하며, 브로드캐스트 주소는 다음 네트워크 ID의 이전 번호가 됩니다.
